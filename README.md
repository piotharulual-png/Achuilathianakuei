# SDG 4 - Quality Education Website

A comprehensive, interactive website dedicated to **Sustainable Development Goal 4: Ensuring Quality Education for All**.

## 📋 Project Overview

This website addresses the global challenge of unequal access to quality education and presents **EduConnect**, an innovative online learning platform as a solution.

### The Problem
- **258 million** children out of school globally
- **617 million** youth lacking basic skills
- **4 billion** people without internet access
- Limited access to qualified teachers
- Financial barriers preventing education access
- Digital divide in learning opportunities

### The Solution
**EduConnect Platform** - A free, accessible online learning platform that:
- Provides global access to quality education
- Offers expert-created courses for all ages
- Issues recognized certificates for career advancement
- Builds a supportive learning community
- Tracks progress with personalized recommendations
- Remains 100% free for all learners

## 🚀 How to Use

1. **Open the Website**
   - Simply open `index.html` in any modern web browser

2. **Navigate Sections**
   - Use the sticky navigation bar to jump between sections
   - Click any section link or "Explore Our Solution" button

3. **Explore the Problem**
   - View global education statistics
   - Understand key barriers to education access
   - See progress indicators for current enrollment rates

4. **Discover the Solution**
   - Learn about EduConnect's 6 core features
   - Understand how the platform works

5. **Browse Courses**
   - Filter courses by category (Elementary, Secondary, Skills)
   - View course details and duration
   - Enroll using the enrollment form

6. **View Impact**
   - See animated statistics on platform reach
   - Read testimonials from successful learners
   - Understand the positive change being made
